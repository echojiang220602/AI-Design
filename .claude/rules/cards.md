# 卡片系统约定

## 文件夹结构

- 21-IndexCard-索引卡/ — 主题索引卡，格式 K编号-关键词.md
- 22-BibCard-阅读卡/ — 阅读笔记和参考文献
- 23-MainCard-核心卡/ — 提炼后的核心知识卡片

## 命名规范

- 索引卡：`K编号-关键词.md`（如 K101-个人生产力.md）
- 编号由用户手动指定，**一旦分配不再改变**

## 卡片流转

口述/阅读 → 直接写入 23-MainCard-核心卡/

索引卡（21-IndexCard/）汇总同主题核心卡的双链

## 操作规则

- 创建核心卡：Frontmatter 只写 `创建日期` + `AI备注`，不添加其他属性
- 操作索引卡时：`index` 属性只用 21-IndexCard-索引卡/ 中已有的索引卡双链，不新建
