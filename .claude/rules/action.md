# 任务管理约定

## 文件夹结构

- 11-Focus-聚焦承诺/ — 与 12 周目标相关的核心项目文件夹
- 12-Active-活跃跟进/ — 当前正在跟进的任务
- 13-Maybe-将来也许/ — 暂时搁置但不放弃的任务

## 命名规范

- 任务文件：`T日期-描述.md`（如 T260101-建立第一个12周目标.md）
- 项目文件夹：放在 11-Focus-聚焦承诺/ 下，自由命名
- 日期：YYMMDD（6 位短格式）

## 任务流转

- 新任务 → 12-Active/
- 与 12 周目标强绑定的项目 → 11-Focus/
- 暂时搁置 → 13-Maybe/
