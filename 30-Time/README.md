# 30-Time/ 时间管理

## 结构

- 31-Vision-愿景/ -- 长期愿景和人生方向
- 32-12Week-十二周/ -- 12 周周期的目标和计划
- 33-Weekly-每周/ -- 周日志和周回顾
- 34-Daily-日志/ -- 每日日志

## 命名规范

- 周日志：YYYY-Wxx.md（如 2026-W07.md）
- 日志：YYYY-MM-DD（周几）.md（如 2026-02-16（周日）.md）
- 12 周计划和愿景：自由命名

## 工作流

- 每日：在 34-Daily-日志/ 写日志，模板含"今日重点"字段
- 每日（可选）：`-日常-每日开场` skill 帮助确定今日重点；`-日常-每日收场` skill 做收尾回顾
- 每周：`-日常-周回顾` skill 做对齐检查并写入周记

## 周数与 12 周周期映射

不使用 frontmatter 属性关联，而是通过周数自动推算：
- W01-W12 → Y1，W13-W24 → Y2，W25-W36 → Y3，W37-W52 → Y4
