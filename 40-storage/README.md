# 40-storage/ 资源配置

## 结构

- 41-archive-归档/ -- 已完成或不再活跃的内容
- 42-template-模板/ -- Obsidian 模板文件
- 43-picture-图片/ -- 图片资源
- 44-base-数据视图/ -- Obsidian Bases 数据视图文件（.base 格式）
- 45-config-配置文件/ -- 系统配置

## 常用位置

- 模板文件在 42-template-模板/ 中
- 数据视图在 44-base-数据视图/ 中
