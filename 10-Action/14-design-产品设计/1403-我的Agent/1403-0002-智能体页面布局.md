智能体页面整体按照左右布局的格式


整体：2×2 Grid 布局

┌──────────────────┬──────────────────┐

│ 卡片1 │ │

│ 我的中台助手 │ 人格设定 │ ← 这两格等高

│ +对话统计 │ (Tab卡片) │

├──────────────────┤ │

│ 卡片2 │ │ ← 卡片1+2合并高度 = 人格设定高度

│ Agent实例 │ │

│ +运行状态 │ │

│ +资源使用率 │ │

├──────────────────┼──────────────────┤

│ 定时任务 │ 技能工具列表 │ ← 这两格等高

└──────────────────┴──────────────────┘

  
结构说明：

|区域|内容|高度关系|
|---|---|---|
|左上|卡片1：我的中台助手 + 对话统计|rowspan=2，等于右侧人格设定|
|左中|卡片2：Agent实例 + 运行状态 + 资源使用率|（同上格子的下半部分）|
|右上|人格设定 Tab（人格设定/用户档案/工作手册等）|等高于左侧卡片1+2合并高度|
|左下|定时任务|rowspan=1，等高于右下|
|右下|技能工具列表|rowspan=1，等高于左下|

关键技术点：  

- 左侧前两张卡片（卡片1+卡片2）合并高度 = 右侧人格设定卡片高度
- 用 CSS Grid 的 grid-row: span 2 实现左侧卡片跨两行
- 下方定时任务和技能工具列表左右等高
