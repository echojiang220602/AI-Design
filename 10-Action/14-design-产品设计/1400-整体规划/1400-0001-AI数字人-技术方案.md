### 2.2 基础 UI 组件清单

  

| 组件 | 变体/Props | 说明 |

|------|-----------|------|

| `<Button>` | `variant`: primary / outline / ghost / danger | 渐变 primary（coral） |

| `<Textarea>` | `maxLength`, `showCount` | 同 Input，高度可调 |

| `<Badge>` | `variant`: hot / official / must / new | 不同背景+文字颜色 |

| `<StepIndicator>` | `steps: { label, status }[]` | 三步骤指示器 |

| `<LoadingOverlay>` | `show`, `title`, `sub`, `progress` | 全屏 loading，带进度条 |

| `<EmptyState>` | `icon`, `title`, `sub` | 无数据引导状态 |

| `<UploadZone>` | `accept`, `onFileChange`, `preview` | 拖拽+点击上传 |

| `<Breadcrumb>` | `items: { label, href? }[]` | 面包屑导航 |

## 四、页面详细规格

  

### 4.1 首页 `/`（创建 Agent）

  

**对应静态页：** `AI-virtual-human-platform.html`

  

**布局：** 左右分栏，左侧背景大图+文字卡片浮层，右侧毛玻璃表单卡片

  

```

<Nav />

<main class="flex h-screen">

  <LeftPanel class="flex-0-0-58%">

    <TextCard />

  </LeftPanel>

  <RightPanel class="flex-0-0-400px">

    <StepCard>

      <StepHeader step={step} />

      {step === 1 ? <Step1Form /> : <Step2Form />}

    </StepCard>

  </RightPanel>

</main>

<LoadingOverlay />

```

  

**Step 1 表单字段：**

- 机器人昵称（Input, required）

- Client ID（Input, required）

- Client Secret（Input password, required）

- 验证按钮 → 模拟 1.2s 延迟，成功后启用"下一步"

  

**Step 2 表单字段：**

- 数字人昵称（Input, ≥3字符）

- 我是谁（Textarea, ≥10字符，预填示例内容）

- 它的灵魂SOUL（Textarea, 50~500字，实时字数统计）

  

**Loading 动画 5 阶段（2000ms + random 600ms 间隔）：**

1. 正在连接 Kubernetes 集群… (15%)

2. 正在调度 Pod 资源… (35%)

3. 正在拉取 Docker 镜像… (55%)

4. 正在启动 OpenClaw 容器… (75%)

5. 正在注入灵魂设定… (90%)

6. 🎉 您的专属 Agent 已就绪！(100%)

## 五、状态管理（Zustand）

  

```ts

// stores/agentStore.ts

interface AgentStore {

  step: 1 | 2;

  robotNickname: string;

  clientId: string;

  clientSecret: string;

  verified: boolean;

  agentNickname: string;

  whoAmI: string;

  soul: string;

  setStep: (s: 1 | 2) => void;

  verify: () => Promise<void>;

  createAgent: () => Promise<void>;

  reset: () => void;

}

  

// stores/skillStore.ts

interface SkillStore {

  skills: Skill[];

  packs: Pack[];

  activeFilter: 'all' | 'official' | 'hot' | 'must';

  filteredSkills: Skill[];

  fetchSkills: () => Promise<void>;

  fetchPacks: () => Promise<void>;

  setFilter: (f: FilterType) => void;

}

  

// stores/diaryStore.ts

interface DiaryStore {

  entries: DiaryEntry[];

  currentEntry: DiaryEntry | null;

  loading: boolean;

  showUpload: boolean;

  fetchEntries: () => Promise<void>;

  fetchEntry: (id: string) => Promise<void>;

  createEntry: (form: FormData) => Promise<void>;

  deleteEntry: (id: string) => Promise<void>;

  setShowUpload: (v: boolean) => void;

}

```

第4步  →  开发 Layout 组件（Nav + 全局布局框架）

第6步  →  实现 Home 页（两步表单 + loading）

第7步  →  实现 Diary 页（列表 + 详情 + 上传弹窗）

第8步  →  串联 API（skillStore + diaryStore + agentStore）

第9步  →  对接后端 API（mock → 真实接口）

第10步 →  部署配置（Vite build + 静态部署）
