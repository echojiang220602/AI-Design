- **框架**：React 18 + TypeScript（严格模式）

- **样式**：Tailwind CSS（含 `@tailwindcss/typography` 用于 Markdown 渲染）
- **图标**：Lucide React
- **Markdown 渲染**：`react-markdown`
- **构建**：Vite 5 + pnpm
- **测试**：Playwright + `@playwright/test`

```
├─────────────────────────────────────────────────────────────────┤
│  [基础信息]                                                      │
│  ──────────────────────                                         │

│                                          └──────────┘           │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 面包屑规格

- 背景：`#F0F9FF`（与页面背景一致）
- 左侧图标：`ChevronLeft`，返回 `/skills/market`
- 格式：`技能市场 › 技能名称`

### 4.3 技能信息区规格

**第1行**：图标（48px 渐变背景）+ 名称 + 类型标签（绿色「技能」/ 青色「工具」）

**第2行**：统计数据内联横排，点分隔，无背景
- `⬇ 下载 {installCount}` · `⭐ AI评分 {rating}` · `♥ 收藏 {Math.floor(installCount * 0.3)}`

**第3行**：描述（`line-clamp-2`）+ 右侧「复制prompt」胶囊按钮

### 4.4 复制 Prompt 胶囊按钮规格

- 样式：`rounded-full border-2 border-[#A78BFA] text-[#8B5CF6]`
- hover：`hover:border-[#7C3AED] hover:text-[#7C3AED] hover:bg-[#F5F3FF]`
- 图标：`Copy size={12}`
- 文案：`复制prompt`
- 行为：`navigator.clipboard.writeText(mdContent)`

### 4.5 SKILL.md 渲染规格

- **数据来源**：`fetch(skill.skillMdUrl)` → 读取 `public/skillPackage/*_SKILL.md`
- **Frontmatter 剥离**：正则 `/^---[\s\S]*?---\n?/`
- **Section 分割**：按 `## ` 拆分 Markdown，含 H1 部分映射为「基础信息」
- **Section 标签映射**：

| 原始标签 | 映射后标签 |
|---|---|
| 概述 / 简介 / 介绍 / 关于 / 功能概述 | 基础信息 |
| 使用说明 / 使用指南 / 使用方法 / 快速开始 | 使用说明 |
| 功能 / 功能介绍 / 功能特性 / 核心功能 / 主要功能 | 功能介绍 |
| 版本历史 / 更新日志 / changelog / 更新记录 | 版本历史 |

- **标题样式**：`text-base font-bold text-neutral-800 pb-2 mb-4 border-b border-neutral-200`
- **正文渲染**：`ReactMarkdown` + `@tailwindcss/typography prose prose-sm`
- **重复去重**：`seenLabels` Set 防止同一映射标签重复渲染

### 4.6 悬浮按钮

- 文案：`skill发布`
- 其他同技能市场列表页

## 7. 发布 Skill 表单页

**文件**：`src/pages/shrimp/pages/SkillPublishPage.tsx`

### 7.1 表单字段

| 序号 | 字段 | 类型 | 必填 | 字数限制 | 错误提示 |
|---|---|---|---|---|---|
| 1 | 技能名称 | 单行输入 | 是 | 30字 | 超出字数 / 必填 |
| 2 | 分类 | 下拉选择 | 是 | — | 必填 |
| 3 | 一句话简介 | 多行（3行） | 是 | 50字 | 同上 |
| 4 | 核心功能说明 | 多行（6行） | 是 | 200字 | 同上 |
| 5 | 使用场景 | 多行（4行） | 是 | 200字 | 同上 |
| 6 | 示例提示词 | 多行（10行） | 是 | 20-2000字 | 至少20字 / 不超2000字 |

**分类选项**：研发工具 / 数据分析 / 内容创作 / 运营推广 / 客户服务 / 安全合规

### 7.2 字数实时提示

| 字数状态 | 边框色 | 提示文字颜色 |
|---|---|---|
| 0 | `border-neutral-200` | `text-neutral-400` |
| 1 ~ 80% | `border-neutral-200` | `text-neutral-400` |
| 80% ~ 100% | `border-orange-400` | `text-orange-500` |
| 超出 | `border-red-500` | `text-red-500` |

### 7.3 提交流程

1. 点击「提交技能」→ 展示确认弹窗（ConfirmModal）
2. 确认弹窗显示：技能名称 / 分类 / 一句话简介 + 安全扫描提示
3. 点击「确认提交」→ 1.5s 加载动画 → 展示成功 Toast
4. 成功 Toast 5s 后自动消失

### 7.4 页面 Header

- 背景：`#F0F9FF`（固定，不变白）
- 标题：「发布技能」
- 副标题：「填写技能信息，提交后系统将自动进行安全扫描」
