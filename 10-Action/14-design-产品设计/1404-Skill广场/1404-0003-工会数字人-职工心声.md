## 一、问卷（wjx-cli-use）

### 1.1 wjx-cli-use

**功能提炼**: 问卷星 OpenAPI CLI，支持问卷/投票/考试/表单创建、答卷管理、数据分析
**数据来源**: https://skillhub.cn/skills/wjx-cli-use
**适用场景**: 问卷调查、数据收集、市场调研、客户满意度、NPS 分析

#### 安装提示词

> 请先检查是否已安装 SkillHub 商店，若未安装，请根据 https://skillhub.cn/install/skillhub.md 安装 SkillHub 商店，但是只安装 CLI，然后安装 wjx-cli-use 技能。若已安装，则直接安装 wjx-cli-use 技能。

#### 配置 API Key

1. 打开：`https://www.wjx.cn/weixinlogin.aspx?redirecturl=%2Fnewwjx%2Fmanage%2Fuserinfo.aspx%3FshowApiKey%3D1`
2. 微信扫码登录后复制 API Key
3. 执行：`wjx init --api-key <key>`

#### 核心能力

1. **问卷类型全覆盖**：调查、投票、考试、表单（`--type 1/3/6/7`）
2. **JSONL 创建**：70+ 题型支持，矩阵/比重/滑块/文件上传/排序
3. **答卷管理**：查询、实时查看、下载（CSV/SAV/Word）
4. **数据分析**：NPS/CSAT/CES 公式、异常检测
5. **通讯录管理**：联系人、部门、标签、子账号

#### 快速路由

| 用户意图 | 命令 |
| --- | --- |
| 做调查/问卷 | `wjx survey create-by-json --file survey.jsonl` |
| 做考试/测验 | `wjx survey create-by-json --file exam.jsonl --type 6` |
| 做投票 | `wjx survey create-by-json --file vote.jsonl --type 3` |
| 做表单/报名表 | `wjx survey create-by-json --file form.jsonl --type 7` |
| 查看问卷列表 | `wjx survey list` |
| 导出答卷数据 | `wjx response download --vid <vid>` |
| 分析 NPS | `wjx analytics nps --scores "[9,10,7,3]"` |

#### 核心规则

1. **强制使用 `create-by-json`**：禁止 `create-by-text`（已弃用、题型不全、易被 shell 转义破坏）
2. **一个需求 = 一个问卷**：所有题目一次创建
3. **面向用户说自然语言**：不在回复中展示 CLI 命令
4. **API Key 未配置时停止操作**：提醒用户配置后再继续

#### 你可以这么说

> "帮我做一份客户满意度调查，包含 NPS 评分题"
> "出一套 JavaScript 基础测验，10 道选择题"
> "分析一下这组 NPS 评分：9,10,7,3,8,10,6"
> "把问卷 12345 的答卷数据导出为 CSV"

## 二、网页PPT（guizang-ppt-skill）

### 2.1 guizang-ppt-skill

**功能提炼**:横向翻页 HTML PPT，含 WebGL 背景、章节幕封、数据大字报、图片网格。两种风格：① 电子杂志 × 电子墨水 ② 瑞士国际主义
**数据来源**: https://github.com/op7418/guizang-ppt-skill
**适用场景**: 分享演讲、发布会、公众号封面、技术分享

#### 安装提示词

> 请先检查是否已安装 SkillHub 商店，若未安装，请根据 https://skillhub.cn/install/skillhub.md 安装 SkillHub 商店，但是只安装 CLI，然后安装 guizang-ppt-skill 技能。若已安装，则直接安装 guizang-ppt-skill 技能。

#### 核心能力

1. **双视觉系统**：Style A 电子杂志风（衬线+流体背景+暖色）+ Style B 瑞士国际主义（网格+高饱和锚点色）
2. **横向翻页**：键盘 ← → / 滚轮 / 触屏滑动 / 底部圆点 / ESC 索引
3. **Style A 10 种布局**：封面、章节、数据大字报、图文、图片网格、Pipeline、对比等
4. **Style B 22 种锁定版式**：Cover、Statement、KPI Tower、Loop Diagram、Duo Compare 等
5. **主题色预设**：Style A 5 套电子墨水主题，Style B 4 套瑞士锚点色（IKB/柠檬黄/柠檬绿/安全橙）
6. **多平台封面**：同一套视觉规则生成公众号 21:9、1:1 分享卡、小红书 3:4、视频号横版
7. **低性能静态模式**：按 B 键关闭 WebGL 动画
8. **单文件 HTML**：不需要构建，浏览器直接打开

#### 适合 / 不适合

**✅ 合适**: 线下分享、行业内部讲话、私享会、AI 产品发布、demo day、带强烈个人风格的演讲

**❌ 不合适**: 大段表格数据、培训课件（信息密度不够）、需要多人协作编辑

#### 你可以这么说

> "帮我基于这篇文章生成一份 8 页左右的瑞士风 PPT，需要 3 张配图"
> "帮我把这个产品分析文档做成电子杂志风 PPT"
> "基于这份 PPT 的主题，做两张封面：公众号 21:9 头图和 1:1 分享卡"

#### 触发关键词

- "帮我做一份杂志风 PPT"
- "帮我做一份瑞士风 PPT"
- "生成一个水平滑动展示板"
- "杂志风格的排版演示文稿"
- "电子墨水风格演讲幻灯片"
- "根据这篇文章制作一张 21:9 比例的公众号封面图"
