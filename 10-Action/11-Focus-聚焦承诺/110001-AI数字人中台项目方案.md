## 一、项目需求与验收标准

### 1.1 核心需求

| 页面 | 需求描述 | 优先级 |
|------|---------|--------|
| **创建 Agent** | 用户通过表单创建专属数字人 Agent | P0 |
| **我的 Agent（智能体）** | Agent 管理平台：人格设定、技能管理、定时任务、对话统计 | P0 |
| **我的 Agent（文件夹）** | 文件管理：文件树、预览、编辑、上传下载 | P0 |
| **我的 Agent（终端）** | 命令行交互（xterm.js 集成占位） | P1 |
| **我的 Agent（控制台）** | OpenClaw 控制台（占位） | P1 |
| **我的 Agent（远程桌面）** | VNC/WebRTC 远程桌面（占位开发中） | P2 |
| **我的 Agent（设置）** | Agent 配置项管理 | P0 |
| **使用指南** | PDF 引导文档浏览 | P0 |
| **技能商城** | 技能包/技能浏览、安装 | P0 |
| **技能包详情** | 包详情、提示词复制、返回导航 | P0 |
| **技能详情** | 功能说明、示例展示、提示词复制 | P0 |
| **分享记录** | 日记形式记录与 Agent 的交互 | P0 |

## 二、设计风格与色彩搭配

### 2.1 设计风格

**毛玻璃 + 渐变 + 柔和阴影**

- **主风格**：Glassmorphism（毛玻璃效果）
- **视觉调性**：科技感、专业、柔和、友好
- **圆角**：卡片 18px / 大卡片 24px / 按钮 12px
- **阴影**：多层柔和阴影，带品牌色透明度

### 2.2 色彩系统

```css
/* 主色 - 科技蓝 */
--coral: #0099DD;
--coral-soft: #F0F9FF;
--coral-mid: #0099DD;

/* 背景色 */
--cream: #FFF5F0;
--sand: #00AAEE;

/* 文字色 */
--ink: #1A1210;        /* 主文字 */
--ink-mid: #5C4A3D;    /* 次要文字 */
--ink-light: #8C7A6D;  /* 弱化文字 */

/* 状态色 */
--success: #22C55E;
--error: #EF4444;

/* 字体 */
font-family: 'Noto Sans SC', -apple-system, sans-serif;
```

### 2.3 设计 Token

| Token | 值 | 用途 |
|-------|---|------|
| 卡片圆角 | 18px | 统一圆润感 |
| 主色调 | #0099DD | 按钮、hover、高亮 |
| 毛玻璃 | backdrop-blur: 16px | 导航、卡片背景 |
| 阴影 | 多层 rgba 叠加 | 层次感 |
