# 客户里程碑事件定义

> 版本：v1.0
> 日期：2026-07-02

---

## 一、里程碑事件概念

里程碑事件是指客户生命周期中具有标志性意义的**关键节点**，用于追踪客户转化路径、评估客户价值、预测客户行为。

---

## 二、成交转化类

| 事件 | 描述 | 触发时机 | 优先级 |
|------|------|----------|--------|
| 首次咨询 | 客户首次发起咨询 | 客户主动联系 | P0 |
| 需求确认 | 明确客户需求并确认 | 商务沟通完成 | P0 |
| 方案提交 | 向客户提交服务方案/报价 | 方案发出 | P0 |
| 合同签署 | 客户签订服务合同 | 合同生效 | P0 |
| 首单成交 | 完成第一笔交易 | 付款到账 | P0 |
| 复购 | 客户二次及以上购买 | 持续业务 | P1 |

---

## 三、业务往来类

| 事件 | 描述 | 触发时机 | 优先级 |
|------|------|----------|--------|
| 首航完成 | 第一次运输任务完成 | 货物送达 | P1 |
| 客户投诉 | 客户提出正式投诉 | 投诉提交 | P0 |
| 投诉解决 | 投诉问题处理完毕 | 客户确认 | P0 |
| 账期延长 | 客户获得更长账期 | 审批通过 | P2 |
| 信用评级提升 | 客户信用等级上升 | 评级调整 | P2 |

---

## 四、服务触达类

| 事件 | 描述 | 触发时机 | 优先级 |
|------|------|----------|--------|
| 首次拜访 | 业务人员首次拜访客户 | 拜访记录 | P1 |
| 调研完成 | 客户访谈/调研报告完成 | 报告提交 | P1 |
| 客户转介绍 | 客户推荐新客户 | 转介绍线索 | P1 |
| 活跃度下降 | 客户长时间无业务往来 | 沉默天数超阈值 | P0 |

---

## 五、流失与挽回类

| 事件 | 描述 | 触发时机 | 优先级 |
|------|------|----------|--------|
| 流失预警 | 客户活跃度持续下降 | 指标触发 | P0 |
| 客户流失 | 客户终止合作或长期无往来 | 判定条件 | P0 |
| 流失召回 | 流失客户重新激活 | 再次成交 | P1 |
| 挽回失败 | 召回尝试未能成功 | 判定条件 | P2 |

---

## 六、行业特定类（港口物流）

| 事件 | 描述 | 触发时机 | 优先级 |
|------|------|----------|--------|
| 箱号绑定 | 客户箱号与业务绑定 | 绑定完成 | P2 |
| 专线开通 | 客户专属物流线路开通 | 线路启用 | P2 |
| 大客户认证 | 客户升级为大客户 | 认证审批 | P1 |
| 合作协议续签 | 年度合作协议续签 | 合同到期 | P1 |
| 业务类型扩展 | 从单一业务扩展至多类型 | 业务新增 | P2 |

---

## 七、优先级汇总

### P0（核心必选）

- 首次咨询 → 需求确认 → 方案提交 → 合同签署 → 首单成交
- 客户投诉 → 投诉解决
- 活跃度下降 → 流失预警 → 客户流失

### P1（重要）

- 首航完成、首次拜访、调研完成
- 客户转介绍、复购、流失召回
- 大客户认证、合作协议续签

### P2（可选）

- 账期延长、信用评级提升
- 箱号绑定、专线开通、业务类型扩展
- 挽回失败

## 八、事件属性设计

### 基础属性

```tsx
interface MilestoneEvent {
  id: string;              // 事件ID
  customerId: string;        // 客户ID
  eventType: EventType;     // 事件类型
  eventName: string;        // 事件名称
  eventTime: string;        // 事件时间
  description?: string;     // 事件描述
  source: string;           // 来源系统
  operator?: string;        // 操作人
  metadata?: Record<string, any>;  // 扩展字段
}
```

### 事件类型枚举

```tsx
enum EventType {

}
```

## 十、业务价值

| 价值维度 | 说明 |
|----------|------|
| 转化追踪 | 可视化客户从咨询到成交的完整路径 |
| 价值评估 | 里程碑事件数量与客户价值正相关 |
| 行为预测 | 流失预警提前干预，降低流失率 |
| 运营分析 | 分析各阶段转化率，优化销售流程 |
