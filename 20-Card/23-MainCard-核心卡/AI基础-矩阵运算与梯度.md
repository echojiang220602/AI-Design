# AI基础-矩阵运算与梯度

## 两大核心数学基础

| 基础 | 说明 |
|------|------|
| **梯度** | 优化器更新参数的方向 |
| **矩阵运算** | 神经网络计算的核心 |

在大模型时代，数学要求大幅降低，但**矩阵运算**仍是核心。

## 矩阵乘法（点积运算）

### 定义

矩阵乘法要求：A 的列数 = B 的行数

```
C[0][0] = A[0行] · B[0列]（对应元素相乘后求和）
```

### 神经网络中的应用

| 网络类型 | 矩阵运算位置 |
|---------|------------|
| **全连接层** | 特征向量 × 权重矩阵 + 偏置 |
| **卷积神经网络** | 卷积核与图像区域做矩阵点积 |
| **循环神经网络** | 特征值 × U/V/W 矩阵 |

## 关键挑战：矩阵形状变化

在学习神经网络时，最让人困惑的不是理论本身，而是**矩阵形状的变化规律**：

```
输入 (1×4) × 权重 (2×4) → 输出 (1×2)
```

理解矩阵形状变化是掌握深度学习和大模型开发的关键。

## 核心结论

> 如果要从事大模型开发，最核心的数学基础就是**矩阵乘法运算**。

掌握矩阵乘法的原理和形状变化规律，是理解和应用现代 AI 模型的关键。
