# 20-Card/ 卡片系统

## 结构

- 21-IndexCard-索引卡/ -- K编号-关键词.md 格式，用于主题索引
- 22-BibCard-阅读卡/ -- 阅读笔记和参考文献笔记
- 23-MainCard-核心卡/ -- 经过提炼的核心知识卡片

## 编号规则

- 索引卡：K编号-关键词.md（如 K101-个人生产力.md）
- 编号由用户手动指定
- 编号一旦分配不再改变

## 卡片流转

- 个人思考→ 直接写入 23-MainCard-核心卡/
- 索引卡在 21-IndexCard/ 中汇总同主题的核心卡链接
- 阅读卡在 22-BibCard/ 中记录来源和摘要

## 相关 Skills

- `-笔记-收件箱归档`：将有 Keywords 字段的笔记自动归档到卡片文件夹
- `-笔记-笔记分诊`：分析收件箱未处理笔记的分类
